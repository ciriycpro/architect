# 2. IMAP-триггер на 5458508@mail.ru

Date: 2026-04-27

## Status

Superseded by [DEC-005](0005-microservices.md)

## Context

Документовед должен получать документы от Таирова. Каналы: Полина грузит вручную в Drive, либо Таиров отвечает на письма от Agent Caller.

## Decision

Триггер Документоведа — **IMAP на ящик 5458508@mail.ru** (тот же, с которого Agent Caller отправляет письма Таирову).

## Consequences

**Плюсы:**
- Таиров не меняет привычки — просто отвечает на письма
- Один канал для входящих и исходящих
- Не нужен Drive-доступ Таирову

**Минусы:**
- IMAP-нода в N8N глючит на тяжёлых вложениях
- mail.ru блокирует частые подключения (timeout через час работы)
- Нет контроля над парсингом писем

**Решение проблем:** см. DEC-005 — переход на отдельный mail-service.
